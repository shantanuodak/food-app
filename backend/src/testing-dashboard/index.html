Benchmark Truth Sources

These sources define what the expected calories should be. The parser is judged against them.

USDA

Best for simple foods like coffee, milk, fruit, rice, eggs, and raw ingredients.

FatSecret

Best for prepared, branded, and serving-based foods when FatSecret access is active.

SnapCalorie Curated

Manual truth source for foods you review in SnapCalorie and trust enough for evals.

Current Rule

A case is only trustworthy when it has a usable truth source. If the eval says “Needs benchmark,” that case should not be treated as a parser failure yet.
